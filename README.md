# CockPerskiy — платформа продажи подписок на антивирус

**CockPerskiy** — учебный (студенческий) проект, представляющий собой веб-приложение для выбора, покупки и управления подписками на антивирусное ПО. Проект полностью контейнеризован и состоит из REST API на Laravel 11, статического фронтенда и PostgreSQL в качестве СУБД.

## 🛠️ Стек технологий

| Компонент | Технологии |
|-----------|------------|
| **Backend** | PHP 8.2, Laravel 11, Laravel Sanctum (API-токены) |
| **Frontend** | HTML5, CSS3, JavaScript (vanilla, без фреймворков) |
| **База данных** | PostgreSQL 16 (Alpine) |
| **Веб-сервер** | Nginx + PHP-FPM (в одном контейнере) |
| **Контейнеризация** | Docker, Docker Compose |

## 📦 Функциональность

- Просмотр каталога тарифов без авторизации
- Регистрация и вход (email + пароль)
- Личный кабинет с историей заказов и активными подписками
- Покупка подписки (выбор тарифа → оформление заказа → имитация оплаты)
- Получение лицензионного ключа после покупки
- Отмена активной подписки
- Административная панель (просмотр статистики, управление пользователями и тарифами) – только для роли `admin`
- Система уведомлений (заглушка, отправка отключена)
- Mock-сервис устройств (опционально, для демонстрации активации/отзыва лицензий)

## 📋 Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (с включённой виртуализацией)
- [Git](https://git-scm.com/) (для клонирования репозитория)
- Свободные порты **8080** (веб-приложение) и **8081** (mock-сервис, если запущен)
